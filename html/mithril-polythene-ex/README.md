# Polythene for Mithril - setup example

Minimal setup example how to create a Polythene app.

Variations:

* for ECMAScript2015 / es6
* for ECMAScript 5 / es5

## Init

~~~
npm install
~~~

## Build es6

~~~
npm run build-es6
open es6.html
~~~

## Build es5

~~~
npm run build-es5
open es5.html
~~~
